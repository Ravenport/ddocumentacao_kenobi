Função

A aba de usuários oferece ao utilizador do sistema a capacidade de ver, configurar e alterar usuários do sistema, essa aba conta com um campo para localizar o colaborador, um campo para adicionar um novo usuário e uma tabela com todos os funcionários existentes. Segue abaixo uma imagem dessa interface.

Tabela de usuários

Nessa aba podem ser cadastrados novos usuários do sistema além de permitir busca e edição das configurações de acesso dos colaboradores previamente cadastrados.

Botões de opções e bloqueio de usuários

Tanto o azul como vermelho, os botões no canto direito da tabela servem para verificar e alterar dados dos usuários, o vermelho serve para o bloqueio dos mesmos e o azul para visualização das opções de acesso dos sites da Bomix.

Opções de acesso de usuário

Aqui podem ser alteradas as autorizações do usuário em relação ao acesso aos sistemas internos da Bomix.
