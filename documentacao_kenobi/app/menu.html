Menu

Guia para as funcionalidades da aba menu

Função

O menu tem como objetivo possibilitar o acesso aos dados do usuário conectado, configurações do sistema e ativação do modo escuro.

Menu

No menu temos logo no começo o botão de ativação do modo escuro que após ser ativado se mantém ativo até o logout do usuário, logo abaixo tem informado a coligada ao qual o funcionário pertence e em seguida tem a opção de acessar os dados do perfil do usuário.

Nessa área é possivel alterar o nome, e-mail, foto, senha do usuário visualizar sessões logadas e sair delas.

Configurações

Dentro da parte de configuração você tem acesso a 3 botões: o primeiro "Usuários do Sistema" levará para a aba de usuários (Ver página de usuários)

O segundo botão "Configurações de acesso" levará para uma página aonde existe uma lista com todos os tipos de usuário.

Ao clicar no tipo de usuário uma aba com o nome de todos os funcionários que fazem parte desse tipo de usuário será exibida, podendo ser alterado os acessos do tipo de perfil.

Além disso podem ser criados novos tipos de usuários nesse mesmo local.

O último botão da parte "Configurações de Formulário nos leva a uma página aonde a configuração dos fluxos de aprovação acontecem. Lá podem ser adicionadas novas pessoas ao fluxo assim como também podem ser removidas etapas do mesmo.

Ao clicar em criar tarefa uma nova etapa será adicionada ao fluxo de aprovação, no mesmo formato da foto abaixo.

O ultimo botão do menu é o de saída, para que o usuário possa fazer logout do sistema.
